Morgen Stadt

A European Union funded project

“Morgenstadt: City Insights Innovation Network” (m:ci) is a joint research project facilitated by the German Fraunhofer Society for the Promotion of Applied Research and comprised of stakeholders from Europe and around the world in the field of urban sustainability. A 3D GIS database for analysis and visualization of the Cologne District of Mülheim South was developed. The main goal of this effort is to test the use of GIS technologies for smart cities as support for the implementation of a holistic approach to sustainable urban development in the District of Mülheim South. Project results were presented in GISTalk by me and City of Cologne.

  • Date: 2014-2016
  • Skills: Project Management, 3D, Software Architecture, GIS, BIM, Urban Planning, Virtual Reality, Esri CityEngine, Unity

3D-Building Explorer

Filter specific buildings and floors by attributes with charts

C-Through is a prototype for interactive 3D urban planning analysis on the web based on the ArcGIS JS API 4.3. The application visualizes and analysis 3D features according to user preferences and makes it possible to select and filter specific buildings, floors and units by attributes for further investigation.
